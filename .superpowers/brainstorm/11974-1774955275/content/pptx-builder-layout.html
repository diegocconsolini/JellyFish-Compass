A — Side-by-side: Slide List + Preview

Left panel: template picker at top, draggable slide list below, export button at bottom. Right panel: 16:9 preview of selected slide. Click a slide in the list to see its preview.

Data Flow

Mock Mode (default)

All slide blocks use data from data/mock-data.ts. No token needed. Instant preview.

API Mode (token)

User enters Jellyfish API token. Each slide block calls the relevant endpoint via lib/api-client.ts. Real data in preview + export.

MCP Mode (power users)

User connects to running jellyfish-mcp server. Natural language queries: "Show me team metrics for Q1". MCP handles auth + data transformation.
